# CloudMonitor

Directory Monitoring and HTTP Web Traffic Parsing with Email Notify

## Build Steps
```
cd server
python3 -m venv .
source bin/activate
python3 monitor.py

```

## Built With

* Python 3

## Requirements

* There must be a /Dropbox and /Google Drive folder in the user directory

## Limitations

* UNIX only support
* Tested on macOS Sierra
